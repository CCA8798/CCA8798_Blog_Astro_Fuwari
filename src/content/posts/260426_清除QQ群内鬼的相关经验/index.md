 ## 前言
>&emsp;在许多尤其像MTF群的QQ群内，特别容易出现只看不发，自己鹿完还要反手举报、过河拆桥的下头看黄男。<br>
>这些下头看黄男，往轻了讲，严重影响群内氛围，往重了讲，甚至可以说是在对群内xnn和xyn们的资源完完全全地进行<b>单方面</b>剽窃。<br>
>&emsp;因此，清除群内内鬼，彻底扫除下头看黄男，是每位管理此类QQ群的群主都应该关注的问题。

## 清除QQ群内鬼的步骤

### 1. 制定群规及群昵称规则
&emsp;群规是群内秩序的保障，也是清除内鬼的重要工具。在制定群规时，应规定群员在群昵称内包含自己在社交平台上的ID，以便识别。

### 2. 时刻注意群员的QQ等级
QQ等级由**星星、月亮、太阳、皇冠**这四个基础符号组合而成，用来直观地展示用户的活跃度与“网络资历”。

因此，对于QQ等级低于标准(≤15级)的群员，应尤其注意。

以下提供了一份换算表，以便计算QQ等级：

| 等级符号 | 对应等级 | 换算关系 |
| :--- | :--- | :--- |
| 1颗星星 (★) | 1级 | 基础单位，1级1颗星。 |
| 1个月亮 (☽) | 4级 | 4颗星合成1个月亮。 |
| 1个太阳 (☀) | 16级 | 4个月亮合成1个太阳。 |
| 1个皇冠 (♕) | 64级 | 4个太阳合成1个皇冠。 |

### 3. 本地发言记录
若在查看QQ群成员资料卡时发现其提示“无本地发言记录”的字样时，应尤其注意。

:::CAUTION
<b>“无本地发言记录”不代表该群员未曾发言过</b>，该字样同样可以见于以下情况：
- 群员在加入群聊后，从未发言过。
- 群员在加入群聊后，发言过，但发言记录已被删除。
- 你加入群聊的时间比群员最近一次发言的时间更晚。
:::

### 4. QQ头像&昵称&点赞数
对于QQ头像异常、昵称诡异、点赞数少的群员，应尤其注意。

### 5. 群内发言内容
如果你看见有下头看黄男直言不讳的发“想看我🦌的加我”这样的话，请你直接把他飞掉lol

### 6. 对于学生群员
一般地，对国内大多数寄宿学生而言，发言频率较低(尤其是在工作日/周六早上/周日晚上)是极为正常且普遍的。

因此，在开展清君侧时，应尤其注意时间，不要在工作日开展清君侧，要留有冗余。
