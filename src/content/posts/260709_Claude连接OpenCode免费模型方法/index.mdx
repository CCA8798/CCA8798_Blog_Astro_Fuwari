<h4 class="text-lg font-bold" id="安装Claude CLI">安装Claude CLI</h4>
1. 按Win+R输入cmd或PowerShell
2. 输入以下命令安装Claude CLI (cmd/PowerShell有不同安装方式)

cmd:
```cmd
curl -fsSL https://claude.ai/install.cmd -o install.cmd&&install.cmd&&del install.cmd
```

PowerShell:
```powershell
irm https://claude.ai/install.ps1 | iex
```

完成后输入以下命令检查安装是否成功(应该输出版本号)
```bash
claude --version
```

<h4 class="text-lg font-bold" id="安装opencode-free-proxy">安装opencode-free-proxy</h4>

1. 先CD到想安装到的目录(请确保目录存在)

```bash
cd (此处替换为你想安装到的目录)
```

2. 开始安装

:::TIP
- 此后每次重启电脑使用该连接方法都需要按照步骤启动服务
- 可用pm2管理自启动
:::

分别运行以下命令：

(1) 克隆仓库
```bash
git clone https://github.com/bigdata2211it-web/opencode-free-proxy.git
```
(2) 进入仓库目录
```bash
cd opencode-free-proxy
```
(3) 安装依赖
```bash
npm install
```
(4) 启动
```bash
node server.mjs
```
:::WARNING
不要关掉运行窗口(否则服务会直接终止，且后续步骤还需要用到此窗口)
:::

<h4 class="text-lg font-bold" id="配置CC Switch">配置CC Switch</h4>
1. 打开CC Switch主界面
2. 点击上方选择栏，左侧是CLI，右侧是桌面版，按需选择
3. 点击右上角加号

7. 请求地址同样按照提示信息填写，但是要加上后缀“/v1”(如图)
